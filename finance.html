Finance & Stock Portfolio

Brief

As part of Harvard CS50: Introduction to Computer Science, a finance web-app created in Flask via which users can manage portfolios of stocks. This tool allows users to buy stocks at their current portfolio prices. Users can sell their stocks for profit or loss. The IEX Cloud API can be queried to gather stock prices.

Features

Flask micro-framework was used with Python to design a full stack web-application where users can manage their stock portfolio, buy & sell stock, & view real-time stock information. This was possible with aid of the massive database from IEX Cloud API.

Users can create accounts & log in securely. Passwords are hashed & stored securely to protect user data. CSRF protection ensures security requirements are met.

HTML & CSS managed the user-interface & style of the website, ensuring seamless compatibility on mobile & desktop devices.
