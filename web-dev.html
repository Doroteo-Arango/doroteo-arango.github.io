Flask Fitness & Health Companion

Brief

In the age of heart disease, chronic illnesses, and lack of exercise, staying active is essential for prolonging human life. In this Flask web application, fitness enthusiasts are given a tool to track their progress, learn about muscle hypertrophy, and calculate their Body Mass Index (BMI). This web application focuses on one of the larger niches of the exercise communityâ€”those who train for strength and hypertrophy. The application is very user-friendly, whether the user is a novice or an adept lifter. By providing users with an easy-to-use fitness companion, this web application aims to be an indispensable tool for exercise training.

Features

Implementation of a user-registration & authentication system using Flask-Login & SQLAlchemy for database management.

A notes-style progress tracker with information selective to each user created in Python with the option of deleting & adding new notes.

Develop a comprehensive guide on training each major muscle group using HTML & CSS for structuring & styling.

Creation of a body-mass-index calculator using Python & JavaScript for user self-assessment.
